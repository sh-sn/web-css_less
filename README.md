# 🚀 HTML & CSS: Фундамент веброзробки

### Навчальний проєкт для учнів 10 класу

Цей репозиторій є комплексним навчальним модулем, розробленим для того, щоб познайомити учнів із основами створення сучасних вебсайтів. Проєкт поєднує теоретичну базу, практичні завдання, лабораторні роботи та інструменти для самоперевірки.

---

## 📖 Про проєкт

- **Тема уроку:** Основи мов розмітки HTML та таблиць стилів CSS.
- **Цільова аудиторія:** Учні 10-х класів (рівень початківець).
- **Мета уроку:**
  - Сформувати розуміння структури вебдокумента.
  - Навчити базовим тегам HTML5 та семантичній розмітці.
  - Опанувати основні властивості CSS для стилізації тексту та блоків (Box Model).
  - Розвинути навички самостійної розробки нескладних інтерфейсів.

---

## 📂 Структура репозиторію

Проєкт організований за чіткою ієрархією для зручної навігації:

```text
project-name/
├── README.md               # Опис проєкту та інструкції
├── index.html              # Інтерактивний лендинг уроку
├── theory/                 # Теоретичні матеріали
│   ├── main-content.md     # Основна теорія
│   ├── examples.md         # Приклади коду
│   └── glossary.md         # Словник термінів
├── practice/               # Практична частина
│   ├── tasks.md            # Короткі вправи
│   ├── labs.md             # Лабораторні роботи
│   └── projects.md         # Теми фінальних проєктів
├── tests/                  # Контроль знань
│   ├── self-check.md       # Тест для самоперевірки
│   └── assessment.md       # Підсумкова контрольна робота
├── resources/              # Додаткові матеріали
│   ├── images/             # Графіка
│   ├── videos/             # Відео-уроки
│   └── links.md            # Корисні посилання
├── styles/
│   └── custom.css          # Стилі для головної сторінки
└── scripts/
    └── interactive.js      # Інтерактивні функції та анімації
```

## 🛠️ Технологічний стек

Для створення цього уроку та головної сторінки було використано:

- **HTML5** (Семантична розмітка).
- **CSS3** (Glassmorphism, Flexbox, Keyframe Animations).
- **JavaScript ES6+** (Динамічне створення елементів, плавній скрол).
- **Markdown** (Документація та навчальні матеріали).

## 🤖 Використані AI-інструменти

Цей освітній проєкт був створений за допомогою передових технологій штучного інтелекту, що дозволило оптимізувати структуру та забезпечити високу якість контенту:

1.  **Claude (Anthropic)** — розробка структури уроку, створення глосарію та методичних рекомендацій.
2.  **Grok (xAI)** — генерація креативних ідей для практичних завдань та інтерактивних елементів.
3.  **AI Studio (Google / Gemini)** — технічна перевірка коду, глибинний аналіз теоретичних модулів та написання скриптів.
4.  **DALL-E / Microsoft Designer** — розробка візуальних концепцій та промтів для ілюстрування Box Model та вебдизайну.

## 🚀 Як працювати з курсом

1.  **Початок**: Відкрийте файл `index.html` у вашому браузері. Це головна інтерактивна панель уроку.
2.  **Навчання**: Пройдіть послідовно модулі у папці `theory/`.
3.  **Практика**: Виконайте вправи з `practice/tasks.md` та одну з лабораторних робіт.
4.  **Перевірка**: Протестуйте свої знання у розділі `tests/`.
