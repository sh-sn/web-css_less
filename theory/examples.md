# Практичні приклади: HTML та CSS

У цьому розділі зібрані базові та просунуті приклади використання тегів та властивостей, які ми вивчили.

---

## 1. Робота з текстом (HTML)

Різні рівні заголовків та списки допомагають структурувати інформацію.

```html
<!-- Приклад заголовків та абзаців -->
<h1>Головний заголовок сторінки</h1>
<p>
  Це звичайний текст, де ми можемо виділити
  <strong>важливе слово</strong>
  або
  <em>курсив</em>
  .
</p>

<!-- Маркований список -->
<h3>Мої цілі на цей курс:</h3>
<ul>
  <li>Вивчити структуру HTML</li>
  <li>Навчитися стилізувати елементи</li>
  <li>Створити свій перший міні-проєкт</li>
</ul>
```

---

## 2. Робота із зображеннями та посиланнями

Зверніть увагу на атрибути `src`, `alt` та `href`.

```html
<!-- Зображення з альтернативним текстом -->
<img src="https://via.placeholder.com/150" alt="Приклад зображення" width="150" />

<!-- Посилання, що відкривається в новій вкладці -->
<p>
  Відвідайте наш
  <a href="https://google.com" target="_blank">корисний ресурс</a>
  .
</p>
```

---

## 3. Селектори CSS: Як звертатися до елементів

Ми можемо змінювати стиль за назвою тегу, за класом або за ідентифікатором (ID).

**HTML:**

```html
<p>Звичайний абзац (синій текст).</p>
<p class="highlight">Абзац із класом (жовтий фон).</p>
<button id="main-btn">Кнопка з ID</button>
```

**CSS (`styles/custom.css`):**

```css
/* Селектор тегу */
p {
  color: blue;
}

/* Селектор класу (через крапку) - можна використовувати багато разів */
.highlight {
  background-color: yellow;
  font-weight: bold;
}

/* Селектор ID (через решітку) - має бути унікальним для одного елемента */
#main-btn {
  padding: 10px 20px;
  background-color: green;
  color: white;
  border-radius: 5px;
}
```

## 4. Демонстрація Box Model (Блокової моделі)

Цей приклад показує, як відступи та рамки впливають на розмір елемента.

**HTML:**

```html
<div class="box-example">Я — контент всередині коробки.</div>
```

**CSS:**

```css
.box-example {
  width: 200px; /* Ширина контенту */
  padding: 20px; /* Внутрішній відступ */
  border: 5px solid red; /* Рамка */
  margin: 30px; /* Зовнішній відступ від інших елементів */
  background-color: lightgrey;
}
```

## 5. Комплексний приклад: Картка профілю

Поєднання HTML та CSS для створення реального елемента інтерфейсу.

**HTML:**

```html
<div class="profile-card">
  <img src="https://via.placeholder.com/80" alt="Аватар" />
  <div class="info">
    <h3>Олександр Вебченко</h3>
    <p>Майбутній Frontend розробник</p>
  </div>
</div>
```

**CSS:**

```css
.profile-card {
  display: flex;
  align-items: center;
  border: 1px solid #ddd;
  border-radius: 10px;
  padding: 15px;
  max-width: 300px;
  box-shadow: 2px 2px 10px rgba(0, 0, 0, 0.1);
}

.profile-card img {
  border-radius: 50%;
  margin-right: 15px;
}

.profile-card h3 {
  margin: 0;
  font-size: 1.2rem;
  color: #333;
}

.profile-card p {
  margin: 5px 0 0;
  color: #777;
}
```

---

## Як користуватися цими прикладами?

1.  Скопіюйте код HTML у ваш файл `index.html`.
2.  Скопіюйте код CSS у ваш файл `styles/custom.css`.
3.  Відкрийте `index.html` у браузері та спробуйте змінити значення (наприклад, колір або розмір), щоб побачити результат.
