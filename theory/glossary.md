# 📖 Глосарій термінів: HTML та CSS

Цей словник допоможе вам швидко згадати значення основних понять, які використовуються у веброзробці.

---

## 🌍 Загальні поняття

- **Браузер** — програма для перегляду вебсторінок (наприклад, Google Chrome, Firefox, Safari). Він зчитує код HTML/CSS і перетворює його на візуальний інтерфейс.
- **Вебсторінка** — окремий документ в інтернеті, написаний мовою HTML.
- **Фронтенд (Frontend)** — усе, що користувач бачить у браузері та з чим може взаємодіяти (візуальна частина сайту).
- **Верстка** — процес створення структури та зовнішнього вигляду вебсторінки за допомогою коду.

---

## 🏗️ HTML (Розмітка)

- **Тег (Tag)** — базова одиниця HTML. Це команда, яка вказує браузеру, як відобразити контент. Пишеться у кутових дужках, наприклад: `<h1>`.
- **Елемент** — конструкція, що складається з відкриваючого тегу, контенту та закриваючого тегу (наприклад, `<p>Текст</p>`).
- **Атрибут** — додаткова характеристика тегу, яка вказується всередині відкриваючого тегу (наприклад, `src` у тегу `<img>` або `href` у `<a>`).
- **Гіперпосилання** — активний текст або зображення, при натисканні на які відбувається перехід на іншу сторінку або файл.
- **Семантична розмітка** — використання тегів, які описують роль контенту (`<header>`, `<article>`, `<footer>`), а не просто його вигляд.

---

## 🎨 CSS (Стилізація)

- **Селектор** — частина CSS-правила, яка вказує браузеру, до якого саме HTML-елемента потрібно застосувати стилі.
- **Властивість** — параметр елемента, який ми хочемо змінити (наприклад, `color`, `width`, `font-size`).
- **Значення** — конкретне налаштування властивості (наприклад, `red` для кольору або `20px` для розміру).
- **Каскадність** — принцип, згідно з яким браузер вирішує, яке правило застосувати до елемента, якщо їх декілька (пріоритетність стилів).
- **Клас (Class)** — ідентифікатор, який можна призначити багатьом HTML-елементам для однакової стилізації. Починається з крапки в CSS (наприклад, `.button`).
- **ID (Ідентифікатор)** — унікальне ім'я елемента, яке використовується для стилізації лише одного об'єкта на сторінці. Починається з решітки в CSS (наприклад, `#main-header`).

---

## 📦 Візуальна модель

- **Box Model (Блокова модель)** — концепція, згідно з якою кожен елемент на сторінці розглядається як прямокутний блок, що має контент, внутрішні відступи (`padding`), рамку (`border`) та зовнішні відступи (`margin`).
- **Padding (Внутрішній відступ)** — простір між контентом елемента та його рамкою.
- **Margin (Зовнішній відступ)** — простір між рамкою елемента та сусідніми елементами.
