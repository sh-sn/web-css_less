# ✅ Тест для самоперевірки

Цей тест допоможе тобі зрозуміти, наскільки добре ти опанував основи HTML та CSS. Відповідай на запитання самостійно, а потім звірся з правильними відповідями в кінці.

---

## Частина 1: Теоретичні питання

**1. Що означає абревіатура HTML?**

- а) HyperText Management Language
- б) HyperText Markup Language
- в) HighText Marking Language

**2. Який тег використовується для створення найбільшого заголовка?**

- а) `<heading>`
- б) `<h6>`
- в) `<h1>`

**3. Як правильно підключити зовнішній файл стилів `style.css`?**

- а) `<style src="style.css">`
- б) `<link rel="stylesheet" href="style.css">`
- в) `<script href="style.css">`

**4. Яка властивість у CSS змінює колір тексту?**

- а) `background-color`
- б) `text-style`
- в) `color`

**5. Що таке Padding у блоковій моделі (Box Model)?**

- а) Зовнішній відступ від інших елементів
- б) Внутрішній відступ від контенту до рамки
- в) Товщина рамки елемента

## Частина 3: Практична відповідність

З’єднайте термін та його призначення:

1. **`href`** — а) Атрибут для вказання шляху до зображення.
2. **`src`** — б) Атрибут для вказання адреси посилання.
3. **`alt`** — в) Атрибут для опису зображення (якщо воно не завантажилось).
4. **`target="_blank"`** — г) Атрибут для відкриття посилання у новій вкладці.

## 🔑 Правильні відповіді та самооцінка

### Відповіді на Частину 1:

1. **б** (HyperText Markup Language)
2. **в** (`<h1>`)
3. **б** (`<link ...>`)
4. **в** (`color`)
5. **б** (Внутрішній відступ)

### Відповіді на Частину 2:

- **Об'єкт А**: Використано атрибут `link` замість `href`, і закриваючий тег не має похилої риски `</a>`.
- **Об'єкт Б**: 1) Після `20` пропущено одиниці виміру (`px`). 2) У CSS використовується двокрапка `:`, а не знак дорівнює `=`.

### Відповіді на Частину 3:

1-б, 2-а, 3-в, 4-г.
