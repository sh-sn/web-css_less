# 📝 Підсумковий тест: HTML та CSS

Ця контрольна робота складається з трьох рівнів складності. Будь ласка, уважно читайте запитання та записуйте відповіді.

---

## Рівень 1: Початковий (Кожна правильна відповідь — 1 бал)

**1. Який символ використовується для позначення закриваючого тегу в HTML?**

- а) `\`
- б) `/`
- в) `!`
- г) `?`

**2. Яка частина HTML-документа містить службову інформацію (назву сторінки, кодування, підключення стилів)?**

- а) `<body>`
- б) `<footer>`
- в) `<head>`
- г) `<section>`

**3. Яка властивість CSS відповідає за колір фону елемента?**

- а) `color`
- б) `background-color`
- в) `border-color`
- г) `fill-color`

**4. Виберіть правильний запис CSS-селектора для класу з назвою "container":**

- а) `#container`
- б) `container`
- в) `.container`
- г) `@container`

- Б) `<a>` — 2. Маркований список

**7. Який атрибут використовується для того, щоб відкрити посилання в новій вкладці браузера?**
