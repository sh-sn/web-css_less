# 🔗 Корисні ресурси та посилання

Цей список допоможе вам знайти відповіді на питання, вивчити нові фішки та отримати ресурси для створення крутих сайтів.

---

## 📚 Документація та навчання

- **[MDN Web Docs (Ukrainian)](https://developer.mozilla.org/uk/docs/Web)** — "Біблія" веброзробника. Найповніша документація про HTML, CSS та JS.
- **[W3Schools](https://www.w3schools.com/)** — Дуже прості та зрозумілі інтерактивні уроки з прикладами, які можна редагувати прямо в браузері.
- **[Codecademy: Introduction to HTML](https://www.codecademy.com/learn/learn-html)** — Безкоштовний інтерактивний курс для початківців (англійською).
- **[Prometheus: Frontend-розробка](https://prometheus.org.ua/)** — Українська платформа з безкоштовними курсами (шукайте основи HTML/CSS).

---

## 🎮 Навчання через ігри

- **[Flexbox Froggy](https://flexboxfroggy.com/#uk)** — Допоможіть жабці дістатися латаття, вивчаючи CSS Flexbox (є українська мова!).
- **[Grid Garden](https://cssgridgarden.com/#uk)** — Навчіться створювати складні сітки (CSS Grid), вирощуючи моркву.
- **[CSS Diner](https://flukeout.github.io/)** — Класна гра для вивчення селекторів CSS.

---

## 🛠️ Інструменти розробника

- **[Visual Studio Code](https://code.visualstudio.com/)** — Найпопулярніший безкоштовний редактор коду.
- **[CodePen](https://codepen.io/)** — Онлайн-редактор, де можна швидко щось протестувати без створення файлів.
- **[Validator W3C](https://validator.w3.org/)** — Сервіс для перевірки вашого HTML-коду на помилки.
- **[Adobe Color](https://color.adobe.com/)** — Інструмент для підбору гармонійних кольорових гам.

---

## 🖼️ Графіка та шрифти (Безкоштовно)

- **[Google Fonts](https://fonts.google.com/)** — Велика бібліотека безкоштовних шрифтів для вашого сайту.
- **[Unsplash](https://unsplash.com/)** — Якісні професійні фотографії, які можна використовувати безкоштовно.
- **[Pexels](https://www.pexels.com/uk-ua/)** — Ще одна база безкоштовних стокових фото та відео.
- **[Font Awesome](https://fontawesome.com/icons)** — Величезна колекція іконок для сайтів.
- **[Flaticon](https://www.flaticon.com/)** — Безкоштовні векторні іконки у форматі PNG та SVG.

---

## 📺 YouTube канали (Українською)

- **[ITVDN](https://www.youtube.com/@itvdn_ua)** — Багато відео-уроків з основ верстки.
- **[FreeCodeCamp (English)](https://www.youtube.com/c/Freecodecamp)** — Якщо ви володієте англійською, це найкраще місце для навчання.

---

## 💡 Порада для навчання

Не намагайтеся запам'ятати все одразу! Веброзробники постійно користуються Google та документацією. Головне — розуміти принцип роботи, а конкретну назву властивості завжди можна підглянути в **MDN**.

---

**Бажаємо успіхів у навчанні!** Якщо ви створили щось цікаве — обов'язково поділіться з іншими! 🚀
