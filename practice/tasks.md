# 🛠️ Практичні завдання: HTML та CSS

Виконуйте завдання послідовно. Кожне наступне завдання додає нові можливості до вашої сторінки.

---

## Завдання 1: Створення "Скелету" сторінки

**Мета:** Навчитися створювати базову структуру HTML-файлу.

1.  У кореневій папці проєкту створіть файл `index.html`.
2.  Додайте базову структуру (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
3.  Встановіть заголовок сторінки (тег `<title>`) як: _"Мій перший веб-проєкт"_.
4.  У секції `<body>` додайте головний заголовок `<h1>` з текстом "Привіт, світе!".

---

## Завдання 2: Робота з контентом

**Мета:** Вивчити основні теги розмітки тексту.

1.  Додайте підзаголовок `<h2>` з назвою "Про мене".
2.  Напишіть короткий абзац `<p>` (2-3 речення) про свої хобі.
3.  Створіть маркований список `<ul>`, у якому перелічіть 3 технології, які ви хочете вивчити.
4.  Додайте одне зображення (`<img>`) на вільну тему та одне посилання (`<a>`) на ваш улюблений сайт.

---

## Завдання 3: Перші кроки в CSS

**Мета:** Підключити стилі та змінити візуальне відображення.

1.  У папці `styles/` створіть файл `custom.css`.
2.  Підключіть цей файл до вашого `index.html` за допомогою тегу `<link>`.
3.  У файлі CSS встановіть:
    - Колір фону для всієї сторінки (`body`).
    - Центрування для заголовка `<h1>`.
    - Змініть колір та шрифт для тексту в абзацах.

---

## Завдання 4: Робота з Box Model (Блокова модель)

**Мета:** Навчитися керувати відступами та рамками.

1.  Створіть у HTML-файлі блок `<div>` з класом `status-card`.
2.  Всередині блоку напишіть текст: "Статус: Вивчаю HTML та CSS".
3.  У CSS для класу `.status-card` додайте:
    - Ширину (`width`) 300px.
    - Внутрішні відступи (`padding`) 20px.
    - Зовнішній відступ (`margin`) 50px зверху.
    - Рамку (`border`) товщиною 2px, суцільну, синього кольору.
    - Колір фону, який відрізняється від фону сторінки.

---

## Завдання 5: Творче завдання (Синтез знань)

**Мета:** Створити міні-візитку.

Використовуючи отримані знання, створіть блок-картку, яка містить:

- Фото (можна використати заглушку `https://via.placeholder.com/150`).
- Ім'я (заголовок `<h3>`).
- Короткий опис.
- Кнопку (посилання, стилізоване під кнопку).

---

## ✅ Критерії успіху (Чек-лист)

- [ ] Файл `index.html` відкривається в браузері без помилок.
- [ ] Використано правильну структуру HTML5.
- [ ] CSS-стилі підключені окремим файлом.
- [ ] Зображення відображається, а посилання працює.
- [ ] Використано різні типи селекторів (тег, клас).
- [ ] Відступи (padding/margin) налаштовані вірно.
