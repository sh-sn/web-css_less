# 🧪 Лабораторні роботи

У цьому розділі представлені великі практичні роботи, мета яких — закріпити навички верстки та стилізації реальних інтерфейсів.

---

## Лабораторна робота №1: Створення персональної сторінки-візитки

**Мета:** Навчитися використовувати HTML для структурування інформації та CSS для створення привабливого дизайну.

### 📋 Завдання:

Створити вебсторінку, яка представляє вас як майбутнього фахівця або розповідає про ваші захоплення.

### 🛠️ Хід роботи:

1.  **Проєктування**: Намалюйте на папері або в графічному редакторі приблизну схему вашої сторінки (де буде фото, де текст, де контакти).
2.  **Розмітка (HTML)**:
    - Використовуйте семантичні теги: `<header>`, `<main>`, `<footer>`.
    - Додайте фотографію (або іконку), заголовок із вашим ім'ям, короткий опис (2-3 абзаци) та список ваших навичок.
    - Додайте розділ "Контакти" з посиланнями на соцмережі або пошту.
3.  **Стилізація (CSS)**:
    - Виберіть приємну кольорову гаму (фоновий колір та колір тексту).
    - Налаштуйте шрифти (використовуйте `font-family`).
    - Використовуйте **Box Model**: додайте відступи (`padding`) всередині блоків та `margin`, щоб відокремити елементи один від одного.
    - Зробіть рамку або тінь (`box-shadow`) для головної картки з контентом.

### 🏁 Результат:

Готова сторінка `index.html`, яка виглядає як професійна візитка.

---

## Лабораторна робота №2: Створення інформаційної сторінки (Landing Page)

**Мета:** Опанувати роботу з групуванням елементів та створенням структури багатосекційного сайту.

### 📋 Завдання:

Створити односторінковий сайт на тему "Моя улюблена технологія" (або хобі, книга, фільм).

### 🛠️ Хід роботи:

1.  **Шапка сайту (`<header>`)**:
    - Створіть навігаційне меню (`<nav>`), яке містить 3 посилання (наприклад: Головна, Галерея, Про автора).
2.  **Основний контент (`<main>`)**:
    - **Секція 1 (Hero)**: Великий заголовок, короткий слоган та кнопка "Дізнатися більше".
    - **Секція 2 (Деталі)**: Текстовий опис теми, розділений на два стовпчики або блоки.
    - **Секція 3 (Галерея)**: Набір із 3-4 зображень з підписами (`<figure>` та `<figcaption>`).
3.  **Стилізація**:
    - Зробіть навігацію фіксованою або просто стилізуйте її як горизонтальний список (`display: inline-block` або `display: flex`).
    - Налаштуйте стилі для кнопок: зміна кольору при наведенні курсору (`:hover`).
    - Використовуйте властивість `max-width: 1000px` та `margin: 0 auto` для центрування всього контенту на великих екранах.

---

## 📝 Вимоги до звіту про виконання:

Для кожної лабораторної роботи учень має надати:

1.  **Вихідний код** (файли `.html` та `.css`).
2.  **Скріншот** фінального вигляду сторінки в браузері.
3.  **Короткий висновок**: що було найскладнішим, які нові теги або властивості були використані.

---

## 🌟 Додаткове завдання для "профі":

Спробуйте зробити вашу сторінку адаптивною (щоб вона виглядала добре на мобільних телефонах), використовуючи відсоткові значення ширини (`width: 90%`) замість пікселів.
